# 资金面 / 筹码层

使用场景：两融、大宗交易、股东户数、分红送转、120 日资金流。

## 函数

| 需求 | 函数 | 数据源 | 说明 |
|---|---|---|---|
| 融资融券 | `margin_trading(code, page_size=30)` | 东财 datacenter | 日级融资余额、融资买入、融资偿还、融券余额。 |
| 大宗交易 | `block_trade(code, page_size=20)` | 东财 datacenter | 成交价、量、金额、买卖方营业部、溢价率。 |
| 股东户数 | `holder_num_change(code, page_size=10)` | 东财 datacenter | 季度股东数、环比变化、户均持股。 |
| 分红送转 | `dividend_history(code, page_size=20)` | 东财 datacenter | 派息、送股、转增、进度。 |
| 120 日资金流 | `stock_fund_flow_120d(code)` | 东财 push2his | 主力/大单/中单/小单日级净流入。 |

## 解读口径

- 融资余额上升：杠杆资金参与度提高，不等同于基本面改善。
- 大宗交易折价：可能表示流动性折价或股东减持压力，需要结合公告。
- 股东户数下降：可作为筹码集中线索，但不是主力吸筹的直接证据。
- 分红送转：注意税前/税后、除权除息日和实施进度。
- 120 日资金流：适合看持续性；分钟资金流适合看当日异动。

## Caveat

这些端点均依赖东财，批量调用时必须串行并调大 `EM_MIN_INTERVAL`。
